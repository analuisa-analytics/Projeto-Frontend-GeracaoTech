# Projeto Frontend GeracaoTech

## Descrição
O **Projeto Frontend GeracaoTech** é uma aplicação web construída com **React** e **Vite**, fornecendo uma interface moderna e responsiva para interação com funcionalidades de geração tecnológica. O projeto utiliza **Tailwind CSS** para estilização e está preparado para desenvolvimento rápido com hot-reload e boas práticas de linting.

---

## Estrutura do Projeto
Projeto-Frontend-GeracaoTech/
│
├─ public/ # Arquivos estáticos (index.html, imagens, etc.)
├─ src/ # Código-fonte da aplicação
│ ├─ components/ # Componentes React reutilizáveis
│ ├─ pages/ # Páginas da aplicação
│ └─ main.jsx # Ponto de entrada da aplicação
│
├─ .gitignore # Arquivos e pastas ignoradas pelo Git
├─ package.json # Gerenciador de dependências e scripts
├─ package-lock.json # Trava de versões de dependências
├─ vite.config.js # Configuração do Vite
├─ tailwind.config.js # Configuração do Tailwind CSS
├─ eslint.config.js # Configuração do ESLint
└─ index.html # Template principal da aplicação


---

## Requisitos

Antes de executar o projeto, certifique-se de ter instalado:

- **Node.js** v18 ou superior
- **npm** ou **yarn**
- Navegador moderno (Chrome, Firefox, Edge, etc.)

Dependências principais:

- React 18+
- Vite 4+
- Tailwind CSS 3+
- ESLint

---

## Instalação

1. Clone o repositório:
```bash
git clone https://github.com/analuisa-analytics/Projeto-Frontend-GeracaoTech.git
```
2. Acesse a pasta do projeto:
cd Projeto-Frontend-GeracaoTech
3. Instale as dependências:
npm install
4. Inicie o servidor de desenvolvimento:
npm run dev
5. Abra o navegador e acesse:
http://localhost:5173

## Uso

Após iniciar o servidor de desenvolvimento, a aplicação estará disponível localmente. Você pode:

- Navegar entre páginas configuradas no React Router
- Interagir com componentes reutilizáveis
- Visualizar alterações instantaneamente graças ao Hot Module Replacement (HMR) do Vite
- Funcionalidades Principais
- Interface moderna com React + Tailwind CSS
- Configuração de rotas usando react-router
- Boas práticas de linting com ESLint
- Ambiente de desenvolvimento rápido com Vite HMR
- Estrutura modular de componentes e páginas para fácil manutenção
